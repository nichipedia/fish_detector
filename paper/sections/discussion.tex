Several approaches were attempted to improve prediction.
Training epochs were increased, but the model converged quickly to a loss of 0.2 regardless of how many epochs were used.
Different network architectures were attempted. Adding nodes and changing the activation function did not yield better results.
Changing the learning rate created fluctuation in the loss graph, but it also did not help with prediction. 
Adam was attempted instead of SGD but this did increase prediction results. 
Additional samples were also added, but this did not give a better prediction result.
