\section{Results}
The first part of this project was to develop a model that classifies if a fish is present in a image. 
Using the default configurations in PyTorch this was acheived in short time.
The second part was to develop a model that could identify the kind of fish or if there was no fish. 
This proved to be much more challenging. 
Tables \ref{tab:binarymetrics} and \ref{tab:multimetrics} have the results for each model.

Each model trained in this project were trained using the default transforms from PyTorch. I found these to work very well out the box. Espicially for the binary problem.
The default transforms are a Tesnor Conversion, a image resize to 256x256, a center crop, a float32 conversion, and a channel normalization.
Below is a representation of the transforms used in python

\begin{lstlisting}[language=python]
transform = T.Compose([
    T.Resize(256),
    T.CenterCrop(224),
    T.ToTensor(),
    T.Normalize(mean=[0.485, 0.456, 0.406],
                std=[0.229, 0.224, 0.225]),
])

\end{lstlisting}
The binary model was trained with a batch size of 128. 
It used CrossEntropy for its loss function and a learning rate of 0.004 with a Adam optimizer.
I found these parameters to work espicially well for the binary classification problem.
The multi fish classification models also used CrossEntropy for its loss function. The Adam Optimizer was used with a learning rate of 0.004.
Each model required a different batch size because of memory constraints.
Table \ref{tab:batchsize} shows the batch sizes used.
The binary model converged in 12 epochs. The multi fish classification models converged in 20 epochs.

\end{table}


\subsection{Binary ResNet18 Model}
The first model ran was the binary classification case. A Res18 model was trained using transfer learning and the default PyTorch transforms. 

The ResNet 18 model performed very well with the default configurations. Table \ref{tab:binarymetrics} has the results for the performance of the model.
The confusion matrix, ROC graph, and loss graph can be seen at \ref{fig:binaryconf} \ref{fig:binaryroc} \ref{fig:binaryloss} respectively.
Achieving a 98\% accuracy was a very strong performance. This lead me to immediatetly attempting the multi fish problem.
