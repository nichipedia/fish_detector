\section{Introduction}

Accurate fish classification is important for biodiversity and monitoring. Significant work has been performed by USM students and faculty to create models to automate fish detection. This project is to reimplement convolutional neural network (CNN) models that have demonstrated success. In so doing, gaining experience with deep learning for image classification was recently implemented to solve a real-world problem.
To do this, two sets of models are implemented. The first set containing a model that is trained to make a true false(binary) prediction on if a fish is present in a image.
The second set being a collection of models trained to identify the species of fish or if the fish isn't present at all.
